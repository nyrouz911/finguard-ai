# Use Case — FinGuard AI

## Objective
Build a system to:
- Predict customer churn
- Detect risky/fraud behavior
- Analyze customer feedback

## Business Questions

### Churn
- Which customers are likely to leave?
- What behaviors indicate churn?

### Risk
- Which transactions/customers are suspicious?
- What patterns indicate fraud?

### Customer Insights
- What are customers complaining about?
- What sentiment trends exist?

## Outputs
- Churn prediction model
- Risk detection model
- NLP insights from feedback
- Dashboard for decision making
